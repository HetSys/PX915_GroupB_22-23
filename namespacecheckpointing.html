Module contains functions that write checkpoint files and read them in as user input.

Write checkpoint file (netCDF)

Writes netCDF checkpoint file every 'freq_in' timesteps. Checkpoint file contains all user input parameters and concentration up to current timestep.

Parameters
[in]tstepCurrent timestep
[in]tstepsNumber of timesteps
[in]dtTime step size (s)
[in]nNumber of spatial nodes
[in]cConcentration at current timestep (mol m^-3)
[in]DDiffusion constant (m^2 s^-1)
[in]RRadius of simulation sphere (m)
[in]a_smallParticle surface area per unit volume (m^-1)
[in]LElectrode thickness (m)
[in]iappApplied current density (A m^-2)
[in]electrode_chargeElectrode charge, 'p'=positive, 'n'=negative
[in]cstorage2D array containing the concentration profile at each timestep
[in]filenameName of user input file. Used to determin name of checkpoint file
[in]freq_inOptional, interval of timesteps at which to write checkpoints. Default = tsteps/10.
