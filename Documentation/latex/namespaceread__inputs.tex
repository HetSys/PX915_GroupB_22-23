Subroutine reads user inputs from txt file. 

Subroutine reads txt file \textquotesingle{}filename\textquotesingle{}, parsing and returning input parameters. Txt file should contain parameters tsteps, dt, n, c0, D, R, a, L, and electrode\+\_\+charge, in format \textquotesingle{}parameter = value\textquotesingle{}. These parameters can be listed in any order above a line of asterixes $\ast$$\ast$$\ast$. Below the asterix line should be iapp, with a label line followed by an array printed one element per line.
